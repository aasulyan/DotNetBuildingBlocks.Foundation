# DotNetBuildingBlocks.Abstractions

## Purpose

`DotNetBuildingBlocks.Abstractions` contains the smallest shared contracts for the Foundation solution.

This package exists to provide pure reusable interfaces that can be referenced by other low-level packages without pulling in framework-specific or infrastructure-specific dependencies.

## Features

- identity contracts
- basic entity contract
- audit-neutral contract
- soft-delete contract
- minimal result marker contract
- minimal paging query contract

## Installation

```bash
dotnet add package DotNetBuildingBlocks.Abstractions
