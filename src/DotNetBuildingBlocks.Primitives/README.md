# DotNetBuildingBlocks.Primitives

`DotNetBuildingBlocks.Primitives` contains shared low-level primitives for the Foundation solution.

## Included

- `Error`
- `Result`
- `Result<TValue>`
- `ValueObject`
- `StronglyTypedId<TValue>`
- `PagedRequest`
- `PagedResult<T>`

## Notes

This package is framework-neutral and reusable across application layers.
